Human Resources demo application


## How build

Setup java 8 and Maven, see [enviroment_setup.md](enviroment_setup.md) 
  
    
### build project 

Goto Project folder    
    
    mvn clean verify

### Server test

For server test jetty plugin can be used

    mvn jetty:run 
    
Open [http://localhost:8080](http://localhost:8080/)
