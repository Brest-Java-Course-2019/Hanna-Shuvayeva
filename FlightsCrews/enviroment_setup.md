Update packages

    ~$ sudo apt update

Install OpenJDK 8


    ~$ sudo apt install openjdk-8-jdk   
   
Verify

    ~$ java -version

### Install Maven

    ~$ sudo apt install maven

Verify maven installation

    ~$ mvn -version
